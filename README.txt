Book Tracker

Автор: Никитин Тимур

Краткое описание: GUI-приложение для управления списком прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц. Реализована фильтрация по жанру и количеству страниц. Данные автоматически сохраняются в JSON-файл и загружаются при запуске. Присутствует валидация ввода: все поля обязательны, количество страниц должно быть числом.

Зависимости: Стандартная библиотека Python (tkinter, json, os). Никаких сторонних библиотек не требуется.

Инструкция по запуску:

Убедитесь, что установлен Python 3.6 или новее. Проверить: python --version

Сохраните код в файл, например book_tracker.py

Запустите программу командой: python book_tracker.py

Примеры использования:

Добавление книги:

Title: 1984
Author: George Orwell
Genre: Dystopia
Pages: 328
Нажать Add Book — книга появляется в таблице.
Фильтрация по жанру:

Выбрать в выпадающем списке Dystopia — таблица покажет только антиутопии.
Фильтрация по страницам:

В поле Pages > ввести 300, нажать Filter by Pages — останутся книги толще 300 страниц.
Сброс фильтров:

Нажать Show All — отобразятся все книги.
Сохранение:

После добавления книги данные автоматически сохраняются в books.json в папке с программой. Также можно нажать Save вручную.
Обработка ошибок:

Если оставить пустое поле: "All fields must be filled."
Если ввести текст вместо числа страниц: "Pages must be a number."
Если ввести не число в фильтр страниц: "Pages threshold must be a number."
